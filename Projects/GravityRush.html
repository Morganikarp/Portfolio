Gravity Rush

December 2023
Unity
C#
Solo
Gravity Rush was made for my Gameplay Programming module in my 2nd year of university.

The aim was to make a game demo that had a number of destinct complete mechanics. The first of which is a full save and load file system. This required me to learn how to make and intergrate a file manipulation system. The other main mechanic I created was a system that manipulated the Unity physics system using maths to allow the player to swing on a rope and walk up specific surfaces.

This experience was really valuable to me as it was practice in applying more advanced maths to coding. It was a solo project, and so all the programming, art and design was done by me. While it worked as a demo, it unfortunatly is unfinished as a whole game.

This project earned me a 78% for this module, which is equivalent to a first.
