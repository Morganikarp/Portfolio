Fishsplosion
Released april 2024
Unity
C#
Solo
Project Demands
  • Developing a complete product within a week.
  • Designing a simple “fish” game.
Noteworthy Contributions
  • I programmed unique “shooting” mechanics for the two weapons available to the player.
  • Using vector maths, I programmed the player character to constantly rotate to face the player’s chosen angle.
  • I made all the art and animations by myself.
Developed Skills
  • This was my first game that I had put online via WebGL, and so I learned the associated process and prerequisites.
Project Description
Fishsplosion was made for a week-long public game jam called FishFest. It was made in Unity with C#. The game could be anything with a fish aesthetic, so I decided on a simple “shooting” concept. All programming and art was made by me alone. Out of the 562 submissions to the jam, Fishsplosion placed 344th overall.
